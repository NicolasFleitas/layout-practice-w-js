Numbers

Si sumas número y un string el resultado sera un string - Javascript intenta convetir string a numbers en operaciones aritmeticas

Cuidado al sumar numeros, javascript usa el operador + para sumar y concatenar. Numeros suman, strings se concatenan


var x = 10;
var y = "20";
var z = x + y;  // z será 1020 (un string) 



var x = "100";
var y = "10";
var z= x/y; // el resultado sera 10
