2. Extraer texto de un string

Existen 3 metodos

  1. slice(start, end)
  2. substring(start, end)
  3. substr(start, length)

El metodo slice()

slice() extrae una parte de un texto y retorno el texto extraido en un nuevo string

El metodo toma 2 atributos: posicion inicial la posición final (final no incluido).

Si se omite el segundo parametro el metodo cortara hasta el final de la cadena

Substring metodo

El metodo substring es similar a slice

La diferencia es que no acepta valores negativos

El substr() metodo extraer una parte de un string y retorna la parte extraida en un nuevo string

Substr metodo

The substr() metodo extrae una parte de un string y retorna las partes extraida en un nuevo string

La diferencia es ese segundo parametro especificando la longitud del texto a extraer
